## Development Setup

It is recommended to use [VSCode](https://code.visualstudio.com/) + [Volar](https://marketplace.visualstudio.com/items?itemName=Vue.volar).

```sh
npm install
npm run dev    # vite dev server (HMR)
npm run build  # vite prod build
npm run lint   # eslint
```
